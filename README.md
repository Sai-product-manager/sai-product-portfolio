# Sai Chelluri - AI Product Management Portfolio

This is a static portfolio website for Sai Chelluri, focused on AI Product Management, Product Strategy, User Research, UX Evaluation, and Product Analytics.

## Included

- Recruiter-facing landing page
- Product case studies across quick commerce, mobility, B2B SaaS, e-commerce, fitness, and food ordering
- Resume download
- LinkedIn profile PDF
- Downloadable project PDFs and PowerPoint decks

## How to publish with GitHub Pages

1. Create a new public GitHub repository.
2. Upload all files from this folder.
3. Go to repository Settings.
4. Open Pages.
5. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / root
6. Save.
7. GitHub will generate a public portfolio link.

Use that public link in the Featured section and Contact info section of LinkedIn.
